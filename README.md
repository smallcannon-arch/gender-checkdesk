# 校園性平流程法規對照臺

靜態查核工具，用於協助學校承辦人依教育部「校園性別事件通報及調查處理程序參考流程圖（202501修）」查核流程、法規、期限、表件與程序風險。

上線網址：
https://smallcannon-arch.github.io/gender-checkdesk/

## 版本資訊

- 版本：v0.4.x
- 資料基準：教育部流程圖 202501 修
- 最後維護：2026-05

## 系統定位

本工具不是案件管理系統、不是判案系統，也不是完整流程填報系統。

本工具僅作為校園性別事件流程與法規查核輔助；實際案件仍應依現行法規、主管機關函釋及學校性別平等教育委員會決議辦理。

## 技術限制

- 靜態前端
- 不登入
- 不串 Google Sheets
- 不使用 GAS
- 不使用 Cloudflare Worker
- 不提供 AI 問答
- 不儲存個案資料
- 不使用 fetch
- 不使用 localStorage、sessionStorage、indexedDB
